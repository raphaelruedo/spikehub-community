# 🎯 SpikeHub Community Hub

Welcome to the official transparency and feedback center for **SpikeHub**! This repository is the primary channel for the community to interact with the app's development lifecycle.

> **Note:** This repository is used exclusively for product management, bug tracking, and feature suggestions.

---

## 📲 The App

SpikeHub is the ultimate companion for Valorant players, currently serving over 130k active users.

- [Download on Play Store](https://play.google.com/store/apps/details?id=com.raphaelruedo.spike_hub)

---

## 🛠 How to Contribute

### 🐛 Reporting a Bug

Found something wrong? Open an **Issue** using the "Bug Report" template. Please include screenshots and device details to help us investigate.

### 💡 Suggesting a Feature

Have a great idea? Use the "Feature Request" template. We review every suggestion to see how it fits into our roadmap!

### 📝 Patch Notes

To see what's new in every version, check our [Releases](https://github.com/raphaelruedo/spikehub-community/releases) page. We keep detailed logs of technical adjustments and new content.

---

## 🗺 Roadmap

We are currently organizing upcoming major features. Keep an eye on the `planned` and `in-progress` labels in the Issues tab.
